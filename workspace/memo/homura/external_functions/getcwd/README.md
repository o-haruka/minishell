# getcwd - 現在の作業ディレクトリを取得する

## 解説:
getcwdは、現在の作業ディレクトリ（カレントディレクトリ）の絶対パスを取得するための関数です。主にシェルやファイル操作プログラムで、現在どのディレクトリにいるかを知るために使われます。

### SYNOPSIS
    #include <unistd.h>
    char *getcwd(char *buf, size_t size);

### 戻り値について:
- 成功時は、現在の作業ディレクトリの絶対パスを格納したバッファへのポインタを返します。
- 失敗時はNULLを返し、errnoにエラー内容がセットされます。

## バッファサイズの決め方:
[バッファサイズの決め方]([https://](https://chatgpt.com/share/695100ed-4050-8003-a781-270c6a9f640f))
1. 動的割り当て（NULL渡し）⭐️推奨⭐️
   - システムが必要なサイズを自動で決定
   - メモリ効率が良いが、**free()を忘れずに**
2. PATH_MAX を使用する（非推奨）
   - #include <limits.h> で PATH_MAX が定義されている
   - 通常 4096 バイト（Linux/macOS）
3. 固定サイズを使う場合（非推奨）
