# 📦 git stash

## 📦 git stash とは？

一言でいうと： コミットせずに、今の作業状態を「一時退避（引き出しにしまう）」するコマンド。  
(変更がある時にcheckoutができない...いったん片付けよう...)

目的： 書きかけのコードを失わずに、安全に別の作業（ブランチ移動など）を行うため。

## 📝 必須コマンド 4選

### 今の変更を一時退避する

#### とりあえずしまう

```
git stash
```

#### 【✅推奨】 メモ付きで退避する（後で分かりやすい）

```
git stash push -m "メモ"
```

---

### 退避した一覧を見る（何がしまってあるか確認）

```
git stash list
```

---

### 最新の退避した作業を復元する（引き出しから取り出して作業再開！）

```
git stash pop
```

## 💡 4つの「あるある」利用シーン

### 急な割り込み作業！

「別のブランチで今すぐ確認・修正して！」と言われた時。

### ブランチ作り忘れた！

間違えて main など別のブランチでコードを書き進めてしまった時。

(退避 → 新しいブランチを作成して移動 → 復元)

### 一旦やり直したい！

色々いじったけど、全部ボツにして綺麗な状態に戻したい時。

(退避したまま放置しておけばOK)

### コミットを綺麗に分けたい！

複数の機能をまとめて作ってしまったが、別々にコミットしたい時。
